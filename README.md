# RIP

RIP is a library for complex image processing.

## License

GPLv3

## How to use RIP?

Package in your `dub.json`:
```d
   {
      "dependencies": {
        "rip": "~>0.0.1"
      }
   }
```
## Dependencies

RIP needs dlib library for loading jpeg/png/bmp/tga files.
WARNING: image can be saved ONLY in ppm/pam/png formats.
